# Django Blog Application

A full-featured blogging web application built using Python and the Django web framework. This project features secure user authentication, interactive blog post management, and a dynamic database schema to simulate a real-world production environment.

**Project Status:** *In Active Development*  
I am actively building this project to get understanding of backend architecture, database relationships, and secure user sessions in Django.

---

## Features

### Implemented Features
* **User Authentication:** Secure user registration, login, and logout functionalities using Django's built-in authentication system.
* **CRUD Functionality:** Fully operational Create, Read, Update, and Delete operations for blog posts.
* **Relational Database:** Designed a relational database schema utilizing Django's ORM to manage one-to-many relationships (Users to Posts).
* **Dynamic Views:** Utilized Django class-based and function-based views to handle core application logic.

###  Tech Stack & Tools
* **Backend:** Python, Django
* **Database:** SQLite (Development) / PostgreSQL (Planned)
* **Frontend:** HTML5, CSS3, Bootstrap (or whatever styling you are using)
* **Environment:** Python Virtual Environments (`venv`)
